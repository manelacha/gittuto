SASS

SASS est un préprocesseur css qui permet d'utiliser des fonctionnalités qui n'existent pas dans CSS, tout comme les variables, les imbrications, l'héritage ... etc

Instalation
  • Installer ruby
  • Ouvrir la fenetre de commande
  • Lancer la commande gem install sass pour installer sass
Commandes utiles
  • sass fichier.scss fichier.css : la commande permet de compiler le fichier.scss et générer le fichier.css à chaque execution de la commande.
  • sass --watch fichier.scss:fichier.css : l'execution de la commande permettra de compiler le fichier.scss et générer le fichier.css à chaque modification du fichier.scss sans executer la commande à chaque reprise
  • sass dossier_scss:dossier_css la commande permet de compiler tous les fichier.scss du dossier scss et générer tous les fichier.css associés dans le dossier css à chaque execution de la commande.
  • sass --watch dossier_scss:dossier_css l'execution de la commande permettra de compiler tout les fichier.scss du dossier scss et générer tous les fichier.css associés dans le dossier css à chaque modification de l'un de ces fichier.scss sans executer la commande à chaque reprise
Bases SASS
Variables

fichier.scss

$nom_variable : valeur ;

balise / id /classe {
propriété : $nom_variable;
}

fichier.css

balise / id /classe {
propriété : valeur ;
}

Referencement selecteur parent

fichier.scss

$nom_variable : valeur ;

$nom_variable2 : valeur2 ;

balise / id /classe {
propriété : $nom_variable;
& :hover {
propriété : $nom_variable2;
}
}

fichier.css

balise / id /classe {
propriété : valeur ;
}

balise / id /classe:hover {
propriété : valeur2 ;
}

Imbriquations

fichier.scss

balise1 / id1 /classe1 {
balise2 / id2 /classe2 {
propriété : $nom_variable ;
propriété : valeur2 ;
}
}

fichier.css

balise1 / id1 /classe1 balise2 / id2 /classe2 {
propriété : valeur ;
propriété : valeur2 ;
}
Fichiers partielles

Des fichiers scss contenant des extraits de css qu'on importera dans d'autres fichiers scss avec la commande @import

Ces fichiers sont enregistés comme suit : _nomfichier.scss ; le _ permet de savoir que ce fichier est partiel est donc il ne sera pas compilé à un fichier .css

_fichierpartial.scss

balise_par / id_par /classe_par {
propriété : $nom_variable ;
propriété : valeur2 ;
}

fichier.scss

@import '_fichierpartial';

balise / id /classe {
propriété : $nom_variable ;
propriété : valeur3 ;
}

fichier.css

balise_par / id_par /classe_par {
propriété : valeur ;
propriété : valeur2 ;
}

balise / id /classe {
propriété : valeur;
propriété : valeur3 ;
}
Mixins

la commande @mixin permet de créer des groupes css d'une même propriété par exemple on l'applique sur la propriété border-radius en ajoutant les differents prefixes -webkit -moz -ms, apres en fait appel à ce group dans la balise voulu

fichier.scss

@mixin border-radius($radius) {
-webkit-border-radius: $radius;
-moz-border-radius: $radius;
-ms-border-radius: $radius;
border-radius: $radius;
}

border-radius() : c'est le nom du groupe
$radius : c'est le paramettre qui contiendra la valeur voulue

balise / id / classe { @include border-radius(10px); }

la commande @include permet de faire appel au groupe
la valeur 10px sera affecter à la variable $radius;

fichier.css

balise / id /classe {
-webkit-border-radius:10px;
-moz-border-radius: 10px;
-ms-border-radius: 10px;
border-radius: 10px;
}

Heritages

Le SASS permet de créer un ensemble de propriétés dont d'autres blocs peuvent hériter en utilisant la commande @extend

fichier.scss

balise1/id1/classe1 {
propriété: valeur;
}
balise2/id2/classe2 {
@extend balise1/id1/classe1;
propriété2: valeur2;
}

fichier.css

balise1/id1/classe1, balise2/id2/classe2 {
propriété: valeur;
}
balise2/id2/classe2 {
propriété2: valeur2;
}

Exemple
Operateurs

Le SASS permet d'affecter une valeur en pourcentage à une propriété d'un block en ce basant sur des valeurs en pixels en utilisant des operateur(+, -, *, / )

fichier.scss

balise/id/classe {
width: 600px / 960px * 100%;
}

Dans cette exemple on a affecter à width la valeur

de la largeur en pourcentage d'un block ayant une largeur de 600px

contenu dans un block parent ayant une latgeur de 960px;

fichier.css

balise/id/classe {
width: 62.5%;
}

BOWER (Gestionnaire des dépendances)

BOWER permet d'installer les derniéres versions des modules d'un projet avec des simples lignes de commandes dans notre fenêtre de commande.

Instalation
  • Installer NODE.JS
  • Installer GIT
  • Ouvrir la fenêtre de commande
  • Lancer la commande npm install -g bower pour installer bower
  • Lancer la commande bower install nom_module pour installer la derniére version de ce module
  • Lancer la commande bower update nom_module pour mettre à jour la version existante de ce module
  • cette verison sera telecharger dans le dossier bower_components de notre projet
  • les information concernat cette verison seront enregistrées dans le fichier bower.json qui se trouve dans le dossier bower_components/nom_module de notre projet
